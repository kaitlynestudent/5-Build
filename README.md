# 5-Build
Module 5 Build
The Indie Film Collective is a multi-page webpage that describes all of the offered services from the Indie Film Collective. The pages included are the home or index page, the film screenings page, the workshops page, and the collaborations page. The HTML is semantic and validated. The CSS is responsive and the breakpoint is at 800px. There is no javascript.
